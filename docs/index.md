# Bienvenido a la Documentación de CopIA

Esta es la página principal de la documentación del proyecto **CopIA**.

Aquí encontrarás todos los detalles técnicos, diseños arquitectónicos, diagramas de componentes y de despliegue del sistema.

## Estructura de la documentación

Utiliza el menú de navegación lateral para explorar las diferentes secciones:

- **Ingeniería de Software**: Detalles sobre la arquitectura del software y sus componentes.
- **Infraestructura**: Información sobre cómo y dónde se despliega el sistema.

> **Nota:** Este sitio está construido utilizando [MkDocs](https://www.mkdocs.org/) con el tema [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).
